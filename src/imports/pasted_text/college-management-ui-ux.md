Design a modern, scalable, enterprise-level College Management System UI/UX.

Tech Stack Requirement:
- Frontend: React (Vite)
- Styling: Tailwind CSS
- Design Style: Clean, modern, minimal, professional SaaS dashboard
- UI Inspiration: Notion, Stripe Dashboard, Linear App
- Theme: Light + Dark mode support
- Color Palette: Primary (#4F46E5 Indigo), Secondary (#06B6D4 Cyan), Accent (#22C55E Green), Neutral greys
- Typography: Inter / Poppins
- Layout: Fully responsive (Desktop + Tablet + Mobile)

--------------------------------------------------

🎯 SYSTEM OVERVIEW:

Create a complete role-based College Management System with 5 roles:

1. Admin (Super Control)
2. Principal (Full Access)
3. HOD (Department Level Control)
4. Staff (Teaching & Task Management)
5. Student (View & Interaction)

Each role must have separate dashboards, navigation, and permissions.

--------------------------------------------------

🔐 AUTHENTICATION SYSTEM:

Design:
- Login Page (Role-based login selection)
- Register Page (Students & Staff)
- Forgot Password
- OTP Verification UI
- Role Switch UI (if multi-role)

Login Roles:
- Admin
- Principal
- HOD
- Staff
- Student

--------------------------------------------------

📊 GLOBAL DASHBOARD (ROLE BASED):

Each dashboard must include:
- KPI Cards (Total Students, Staff, Attendance %, Results, Fees)
- Charts (Bar, Line, Pie)
- Recent Activities Feed
- Notifications Panel
- Calendar Widget
- Announcements Section
